## COMO EJECUTAR

Primero instala las dependencias

```bash
npm install
```

Luego ejecuta el siguiente comando

```bash
npm run dev
```

ABRIR LA SIGUIENTE URL [http://localhost:3000](http://localhost:3000) EN TU NAVEGADOR FAVORITO.
